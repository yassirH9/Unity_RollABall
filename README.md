# RollABall

## How to install

1] clones the project
  ```sh
git clone https://github.com/yassirH9/Unity_RollABall
  ```
2] open unity hub

3] add project from disk

4] search the project directory and add

5] open project in unity hub

6] press play in unity editor

## Software used

- Unity
- Blender
- Visual Studio 2022
